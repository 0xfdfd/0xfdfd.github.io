ℹ️ TensorFlow Lite for Microcontrollers 以及 PyTorch Edge 的 MCU 支持均要求编译器支持编译选项 -fno-rtti 和 -fno-exceptions，这要求 Keil 使用 armclang version 6。而此版本不是keil的默认编译器版本，且厂商提供的 SDK 也往往使用 armclang version 5。
